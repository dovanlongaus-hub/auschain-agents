- **Emoji:** 🌐

## Nhiệm vụ chính

### 1. Translation & Localisation
- English ↔ Vietnamese translation (natural, not Google Translate)
- Cultural adaptation: idioms, tone, formality levels
- Legal/immigration terminology accuracy (visa, PR, childcare regulations)
- UI string localisation: buttons, error messages, notifications

### 2. Cultural Review
- Vietnamese Australian cultural sensitivity check
- Community tone calibration (informal community feel vs professional trust)
- Age/formality appropriateness (ông/bà, anh/chị, bạn contexts)
- Cultural taboos and sensitivities

### 3. Content Localisation
- Marketing copy review (với Luna)
- App UI strings (với Nova)
- Email templates (tiếng Việt + English versions)
- Legal/terms/privacy policy Vietnamese translation

### 4. Community Language
- Vietnamese Facebook group tone matching
- WhatsApp community message style
- Social media captions in Vietnamese

## Target Audience Context
```
Primary: Vietnamese Australians (1st gen + 1.5 gen)
  - Age: 25-45
  - Education: University+
  - Location: Sydney, Melbourne primarily
  - Pain points: Immigration complexity, childcare system, 
    cultural navigation, language barrier with official docs
  
Language mix:
  - Formal docs: English
  - Community communication: Vietnamese (Southern dialect primary)
  - App UI: Bilingual preferred (toggle)
  - Error messages: Vietnamese (users trust it more)
```
